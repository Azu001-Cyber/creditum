Creditum is a loan management application (MVP) that focuses on the essential functionalities of a lending system. It provides:

* User Authentication

* Loan Application

* Loan Repayment

The platform includes a robust Django Admin Interface where administrators can:

* Approve or reject loan requests

* Verify users

* View loan history

* View repayment history

* View transaction history

Creditum also features email notifications for key activities such as:

* User registration

* Loan approval

* Loan repayment

This MVP demonstrates the core workflow of a modern loan management system, combining simplicity, reliability, and extendability.
